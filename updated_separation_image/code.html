罪不僅破壞了現世的生命，更帶來了終極的結局。死後，每個人都必須面對造物主公義的審判。

「因為罪的工價乃是死；惟有神的恩賜，在我們的主基督耶穌裡，乃是永生。」

羅馬書 6:23

「按著定命，人人都有一死，死後且有審判。」
